# Zomato Data Analysis Dashboard

A simple, interactive Tableau dashboard that looks at Zomato restaurant data to find out what people like, where they spend money, and how restaurants are performing.

---

## 📝 Dashboard Summary
This dashboard takes messy restaurant data and turns it into clean charts. It helps business owners and food analysts instantly see which cuisines are most popular, how much people usually spend, and which cities have the best-rated restaurants. 

---

## 🛠️ Skills Used
* **Data Visualization:** Turning numbers into easy-to-read charts and bars using Tableau.
* **Map Making:** Creating interactive maps to show restaurant prices by location.
* **Data Grouping:** Sorting restaurants into price buckets (like budget vs. luxury) to see where most people eat.
* **Business Insights:** Finding real-world trends from data, like figuring out what customers want next.

---

## 🚀 Dashboard Features
* **Cuisine Popularity Chart:** Shows exactly which food or drink categories get the most votes.
* **City Price Matrix:** A color-coded table showing how many cheap vs. expensive restaurants are in each city.
* **City Price Map:** An interactive map showing the average cost of eating out across different regions.
* **Top Restaurant Leaderboard:** A quick list of the highest-rated and most-voted food chains.
* **Price vs. Rating Graph:** A scatter plot that checks if pricier restaurants actually get better reviews.

---
